## 🧠 Sobre o Projeto

> 💻 Um sistema CRUD desenvolvido em Python rodando direto no terminal  
> ⚡ Focado em lógica, estrutura e evolução como desenvolvedor  

Este projeto utiliza conceitos fundamentais que todo dev precisa dominar:

- 📦 Estruturas de dados (listas e dicionários)
- 🔁 Laços de repetição
- 🔀 Condicionais
- ⚠️ Tratamento de exceções (`try`, `except`, `else`, `finally`)
- 🧩 Organização com funções
